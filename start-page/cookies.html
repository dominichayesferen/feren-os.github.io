Cookies Usage

For the Feren OS Start Page to work properly, this website uses cookies to store the following information offline:

• The search engine you last picked (lastengine) - This is used to remember your last picked Search Engine and choose it whenever the page loads, automatically.

• Your custom background image (userbg) - This is used to remember your custom Start Page background image URL so that the custom background is applied whenever you load the Start Page.

• cookieconsent_status - A cookie from the 'Cookie Consent' box you probably entered this page from to remember if you hit the 'OK' button or not on the prompt

These cookies are stored completely offline, and do not contact the internet at all on the Feren OS Start Page.

More Information:

You can get more information about cookies from the website linked to on the bottom button on the left of the one on the bottom-right, or by visiting https://cookiesandyou.com/, as well as information on how to block cookies in your browser.

Please note that if you block cookies on the Feren OS Start Page then the Settings items and Search Engine Remembering will no longer work.
